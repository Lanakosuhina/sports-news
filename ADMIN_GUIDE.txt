## Вход в систему

1. Перейдите по адресу: `https://ваш-сайт.ru/admin/login`
2. Введите email и пароль
3. Нажмите «Войти»

**Данные для первого входа:**
- Email: `admin@sportsnews.com`
- Пароль: `admin123`

> ⚠️ **Важно:** Смените пароль после первого входа!

## Управление статьями

### Создание статьи

1. Перейдите в раздел **«Статьи»**
2. Нажмите кнопку **«Новая статья»**
3. Заполните поля:

| Поле | Описание | Обязательное |
|------|----------|--------------|
| Заголовок | Название статьи | Да |
| Slug | URL-адрес (генерируется автоматически) | Да |
| Категория | Выберите спортивную категорию | Да |
| Лига | Выберите лигу (если применимо) | Нет |
| Краткое описание | Анонс статьи для списков | Да |
| Содержимое | Полный текст статьи | Да |

### Редактор контента

Редактор поддерживает:
- **Жирный текст** и *курсив*
- Заголовки разных уровней (H1-H6)
- Маркированные и нумерованные списки
- Ссылки
- Изображения
- Цитаты
- Таблицы

### Изображения

- **Главное изображение** — отображается в карточке статьи и в шапке
- **Галерея** — дополнительные изображения внутри статьи

Для загрузки: нажмите «Загрузить изображение» или перетащите файл в область загрузки.

### SEO-настройки

В блоке «SEO» можно указать:
- Meta Title — заголовок для поисковиков
- Meta Description — описание для поисковиков
- Canonical URL — каноническая ссылка

### Публикация

| Статус | Описание |
|--------|----------|
| Черновик | Статья не видна на сайте |
| Опубликовано | Статья доступна всем посетителям |
| Запланировано | Статья опубликуется автоматически в указанную дату |

## Категории и теги

### Категории

Категории — это основные разделы сайта (Футбол, Хоккей, Теннис и т.д.).

**Создание категории:**
1. Перейдите в **«Категории»**
2. Нажмите **«Новая категория»**
3. Заполните:
   - Название
   - Slug (URL)
   - Описание (опционально)
   - Иконка (название иконки из Lucide)
   - Порядок сортировки

### Лиги

Внутри категорий можно создавать лиги (например: РПЛ, АПЛ, Ла Лига).

### Теги

Теги — дополнительные метки для группировки статей.

**Создание тега:**
1. Перейдите в **«Теги»**
2. Нажмите **«Новый тег»**
3. Введите название (slug сгенерируется автоматически)

## Букмекеры

Раздел для управления информацией о букмекерских конторах.

### Создание букмекера

1. Перейдите в **«Букмекеры»**
2. Нажмите **«Новый букмекер»**

### Основные поля

| Поле | Описание |
|------|----------|
| Название | Имя букмекера (например: Winline) |
| Slug | URL-адрес страницы |
| Ссылка на сайт | URL для кнопки «Играть» |
| Домен сайта | Отображаемый домен |
| Бонус | Сумма бонуса (например: 10000 ₽) |
| Метка бонуса | Текст над бонусом (например: Новым игрокам) |
| Рейтинг | От 0 до 5 |
| Мин. депозит | Минимальный депозит |
| Номер лицензии | Номер лицензии ФНС |

### Логотип

Загрузите логотип букмекера:
1. Нажмите **«Загрузить логотип»**
2. Выберите файл с компьютера
3. Или введите URL изображения вручную

### Мобильные приложения

- Отметьте галочками наличие iOS/Android приложений
- Укажите ссылки на App Store и Google Play

### Промо-карточка

Для отображения на страницах бонусов (Фрибет, Без депозита и т.д.):

1. Отметьте галочками страницы для отображения:
   - Фрибет
   - Без депозита
   - Промокод Winline
   - Промокоды Fonbet

2. Заполните промо-поля:
   - Заголовок промо
   - Описание
   - Промокод
   - Срок действия
   - Фоновое изображение

### Изображения страницы

Для страницы букмекера можно загрузить:
- **Фон шапки** — фоновое изображение верхнего блока
- **Баннер мобильного приложения** — изображение в блоке приложений

### Текстовые блоки

Добавляйте уникальный контент на страницу букмекера:
1. Нажмите **«Добавить блок»**
2. Введите заголовок
3. Заполните содержимое в редакторе
4. Блоки можно переупорядочивать стрелками

### Дополнительные поля

Расширяйте информацию о букмекере:
- Время вывода
- Способы оплаты
- Год основания
- Live трансляции (да/нет)
- И другие

## Рекламные блоки

### Создание рекламы

1. Перейдите в **«Реклама»**
2. Нажмите **«Новая реклама»**

### Основные настройки

| Поле | Описание |
|------|----------|
| Название | Внутреннее название баннера |
| Slug | Уникальный идентификатор |
| Размещение | Где показывать баннер |
| Размер | Размер баннера в пикселях |
| Порядок | Приоритет отображения |

### Размещения

| Размещение | Описание |
|------------|----------|
| Шапка сайта | Верхняя часть страницы |
| Сайдбар — верхний блок | Правая колонка, верх |
| Сайдбар — нижний блок | Правая колонка, низ |
| Статья — над контентом | Перед текстом статьи |
| Статья — под контентом | После текста статьи |
| Подвал сайта | Нижняя часть страницы |
| Лента — между статьями | Между карточками новостей |

### Размеры баннеров

| Размер | Название |
|--------|----------|
| 728×90 | Горизонтальный баннер |
| 970×250 | Билборд |
| 300×250 | Средний прямоугольник |
| 300×600 | Полстраницы |
| 120×600 | Небоскрёб |

### Загрузка изображения

1. Перетащите файл в область загрузки
2. Или нажмите для выбора файла
3. Или введите URL вручную

### Внешний рекламный код

Для Google Ads, Яндекс.Директ и других:
1. Вставьте код в поле **«Код рекламы»**
2. Код будет использован вместо изображения

### Расписание показа

- **Дата начала** — когда начать показ
- **Дата окончания** — когда прекратить показ

### Ротация баннеров

Для автоматической смены нескольких баннеров:

1. Выберите **группу ротации** (например: «Сайдбар — верхний блок»)
2. Укажите **интервал ротации** в секундах
3. Все баннеры с одинаковой группой будут чередоваться

### Статистика

Для каждого баннера отображается:
- Количество показов
- Количество кликов
- CTR (кликабельность)

## Управление страницами

Статические страницы сайта (О нас, Контакты, Политика конфиденциальности).

### Создание страницы

1. Перейдите в **«Страницы»**
2. Нажмите **«Новая страница»**
3. Заполните:
   - Заголовок
   - Slug (URL)
   - Содержимое
   - SEO-поля

### Системные страницы

| Slug | Страница |
|------|----------|
| about | О нас |
| contacts | Контакты |
| user-agreement | Пользовательское соглашение |
| privacy-policy | Политика конфиденциальности |
| cookie-policy | Политика Cookie |
| responsible-gaming | Ответственная игра |

## Импорт новостей

Система полуавтоматического импорта новостей из RSS-лент.

### Источники

1. Перейдите в **«Импорт»**
2. Добавьте RSS-источник:
   - Название источника
   - URL RSS-ленты
   - Категория по умолчанию

### Проверка новостей

1. Нажмите **«Проверить»** напротив источника
2. Система загрузит новые материалы
3. Просмотрите список импортированных записей

### Публикация

1. Выберите записи для публикации
2. Нажмите **«Импортировать»**
3. Отредактируйте при необходимости
4. Опубликуйте

### Автоматическая проверка

Система автоматически проверяет источники каждые 30 минут.

- Google Analytics

## Рекомендации

### Изображения

- Используйте формат **WebP** или **JPEG** для фотографий
- Используйте **PNG** для логотипов и иконок
- Оптимальный размер главного изображения: **1200×630 px**
- Сжимайте изображения перед загрузкой

### SEO

- Всегда заполняйте meta title и description
- Используйте ключевые слова в заголовках
- Добавляйте alt-текст к изображениям
- Используйте подзаголовки (H2, H3) в статьях

### Контент

- Пишите уникальные тексты
- Проверяйте орфографию
- Добавляйте релевантные теги
- Используйте актуальные изображения

3. Попробуйте другой браузер
